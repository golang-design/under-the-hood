{{- /* Forward-looking browser-language auto-routing. Behaviour once more than one language has content: * WeChat in-app browser (UA contains MicroMessenger) -> Chinese. * Browser language starting with "zh" -> Chinese. * Everything else -> default (English). * An explicit choice via the language switcher (?lang=) is remembered in localStorage and always wins over auto-detection. It only ever redirects between LANGUAGE HOME roots, never deep article URLs, so a shared/deep link is never bounced. The `homes` map is built from Hugo's home-page translations, so it lists ONLY languages that actually have a home page rendered. Today that is just zh-cn, which means this script is INERT (there is nowhere else to send anyone) and activates automatically the moment an English/German home ships. Runs synchronously, first thing in , so a redirect happens before any paint. */ -}} {{- $homes := dict -}} {{- with .Site.Home -}} {{- $homes = merge $homes (dict .Language.Lang .RelPermalink) -}} {{- range .AllTranslations -}} {{- $homes = merge $homes (dict .Language.Lang .RelPermalink) -}} {{- end -}} {{- end -}} {{- $default := .Site.Params.autoLangDefault | default "en" -}}
